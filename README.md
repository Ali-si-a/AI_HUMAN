# AI_HUMAN

# Оптимизация методов атрибуции авторства для текстов ИИ

Автор: Сибагатова Алина

Год: 2026

## Аннотация

Работа посвящена проблеме распознавания текстов, сгенерированных большими языковыми моделями (DeepSeek, Perplexity, GigaChat), в условиях их стилистической и структурной схожести с человеческими текстами. На материале школьных сочинений (ЕГЭ) разработан и апробирован гибридный подход, объединяющий классические стилеметрические признаки и методы машинного обучения. Наилучшие результаты показал метод опорных векторов (SVM) с RBF-ядром, достигнув Accuracy = 0.953 и ROC-AUC = 0.992.

## Ключевые результаты
*   **Сформирован корпус:** 3685 текстов (человек + ИИ).
*   **Выявлены ключевые признаки:** лексическое разнообразие (MSTTR), доля сочинительных союзов (CCONJ), доля глаголов и местоимений.
*   **Лучшая модель:** SVM с RBF-ядром (C=10, gamma=0.01).
*   **Практическая значимость:** разработаны рекомендации для ручного и автоматического выявления сгенерированных текстов в образовательной практике.
